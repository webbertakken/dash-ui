# Code of conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating, you agree to abide by its terms.

## Enforcement

Reports of unacceptable behaviour can be sent to the maintainer: open a private GitHub issue and tag the repo owner, or email the address listed on the maintainer's GitHub profile.

All reports will be reviewed and investigated promptly and fairly.
