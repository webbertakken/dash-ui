# Dash UI - Svelte

The same 168 components as `@dash-ui/react`, exposed as native Svelte 4 components.

| Package | Description |
|---|---|
| `@dash-ui/tokens` | CSS variables (light + dark motifs) + JS token map |
| `@dash-ui/assets` | Brand SVG logos + product marks |
| `@dash-ui/svelte` | 168 Svelte components |

The React variant lives at the root of this static site; the Web Components variant lives at `/wc/`.
