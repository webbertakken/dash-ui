# Dash UI - Web Components

Each `uni-*` custom element wraps the equivalent `@dash-ui/svelte` component. Importing `@dash-ui/wc` registers all 166 elements as a side effect.

```html
<script type="module" src="https://cdn.example/@dash-ui/wc"></script>
<link rel="stylesheet" href="https://cdn.example/@dash-ui/wc/styles.css" />

<uni-button variant="primary">Save</uni-button>
<uni-pill variant="success">Online</uni-pill>
```

The React variant lives at the root of this static site; the Svelte variant lives at `/svelte/`.
