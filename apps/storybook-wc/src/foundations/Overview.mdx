# Dash UI - Web Components

Each `uni-*` custom element wraps the equivalent `@w5-ui/svelte` component. Importing `@w5-ui/wc`
registers all 166 elements as a side effect.

```html
<script type="module" src="https://cdn.example/@w5-ui/wc"></script>
<link rel="stylesheet" href="https://cdn.example/@w5-ui/wc/styles.css" />

<uni-button variant="primary">Save</uni-button>
<uni-pill variant="success">Online</uni-pill>
```

The React variant lives at the root of this static site; the Svelte variant lives at `/svelte/`.
