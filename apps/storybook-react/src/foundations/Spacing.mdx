# Spacing & radius

The dashboard layout uses an 8px base grid with 4px / 12px / 16px / 24px halves and multiples.
Default radius is 6px.

| Token           | Value  | Use                       |
| --------------- | ------ | ------------------------- |
| `--space-1`     | 4px    | Inline gaps               |
| `--space-2`     | 8px    | Tight grouping            |
| `--space-3`     | 12px   | Card inner padding        |
| `--space-4`     | 16px   | Default gap               |
| `--space-5`     | 24px   | Section gap               |
| `--radius-sm`   | 4px    | Pills, badges             |
| `--radius-md`   | 6px    | Buttons, inputs (default) |
| `--radius-lg`   | 12px   | Modals, drawers           |
| `--radius-pill` | 9999px | Pill capsules             |
