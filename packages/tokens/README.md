# @dash-ui/tokens

Design tokens for the Dash design system. CSS variables + typed JS constants.

```ts
import '@dash-ui/tokens/css';      // imports the CSS variable definitions
import { brand, status } from '@dash-ui/tokens';
```

The CSS exports two motifs:

```html
<html data-motif="dark">  <!-- dashboard -->
<html data-motif="light"> <!-- store / marketing -->
```
